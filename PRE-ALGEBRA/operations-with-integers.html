Operations with Integers

Adding, subtracting, multiplying, and dividing positive and negative whole numbers.

What is an integer?

Integers are the whole numbers and their opposites — no fractions, no decimals — including zero.

The set of integers is written as:

$$\{ \ldots, -3, -2, -1, 0, 1, 2, 3, \ldots \}$$

The number line

Every integer has a place on the number line. Numbers to the right of zero are positive. Numbers to the left of zero are negative.

Key idea — Opposites
Two integers are opposites if they are the same distance from zero but on different sides. For example, $-7$ and $7$ are opposites. The opposite of $0$ is $0$.

Absolute value

The absolute value of an integer is its distance from zero, regardless of direction. It is always positive (or zero).

$$|{-5}| = 5 \qquad |3| = 3 \qquad |0| = 0$$

Adding Integers

When you add integers, the signs tell you what to do. There are only two cases.

Rule 1 — Same signs
If both numbers have the same sign, add their absolute values and keep the sign.
Example 1 — Both positive
$\;\;5 + 3 = 8$
Both positive → answer is positive.
Example 2 — Both negative
$\;\;(-7) + (-4) = -11$
Add 7 + 4 = 11. Both negative → answer is negative.
Rule 2 — Different signs
If the numbers have different signs, subtract the smaller absolute value from the larger one. Keep the sign of the number with the larger absolute value.
Example 3 — Different signs
$\;\;(-9) + 4 = -5$
$|-9| = 9$, $|4| = 4$. Subtract: $9 - 4 = 5$. The $9$ is bigger, and it was negative, so the answer is negative.
Example 4 — Different signs
$\;\;12 + (-5) = 7$
$|12| = 12$, $|-5| = 5$. Subtract: $12 - 5 = 7$. The $12$ is bigger and positive, so the answer is positive.

Practice

1.$(-8) + (-6) = \;?$
Answer:$-14$  (same signs → add 8 + 6, keep negative)
2.$15 + (-9) = \;?$
Answer:$6$  (different signs → 15 − 9, 15 is bigger and positive)
3.$(-3) + 11 = \;?$
Answer:$8$  (different signs → 11 − 3, 11 is bigger and positive)
4.$(-20) + 7 = \;?$
Answer:$-13$  (different signs → 20 − 7, 20 is bigger and negative)

Subtracting Integers

There is only one rule for subtraction — and it turns every subtraction into an addition you already know how to do.

The Golden Rule — Add the opposite
To subtract an integer, add its opposite. Change the subtraction to addition, and change the sign of the second number.

$$a - b \;=\; a + (-b)$$

Example 1
$\;\;7 - 10$
$=\; 7 + (-10)$  (add the opposite)
$=\; -3$  (different signs, 10 is bigger and negative)
Example 2 — Two negatives
$\;\;(-5) - (-8)$
$=\; (-5) + (+8)$  (opposite of $-8$ is $+8$)
$=\; 3$
Example 3
$\;\;(-4) - 6$
$=\; (-4) + (-6)$  (opposite of $+6$ is $-6$)
$=\; -10$
Quick check — Two negatives in a row
A minus sign next to a negative number turns into a plus: $\;a - (-b) = a + b$.

Practice

1.$9 - 14 = \;?$
Answer:$-5$  (rewrite as $9 + (-14)$)
2.$(-6) - 5 = \;?$
Answer:$-11$  (rewrite as $(-6) + (-5)$)
3.$3 - (-7) = \;?$
Answer:$10$  (rewrite as $3 + 7$)
4.$(-12) - (-4) = \;?$
Answer:$-8$  (rewrite as $(-12) + 4$)

Multiplying Integers

Multiplying integers is easy if you remember one short table of signs.

Sign 1Sign 2Product
(+)(+)(+)
(−)(−)(+)
(+)(−)(−)
(−)(+)(−)
In plain English
Same signs → positive.   Different signs → negative.
Example 1
$\;\;(-6) \times (-4) = 24$  (both negative → positive)
Example 2
$\;\;7 \times (-3) = -21$  (different signs → negative)
Example 3 — Three or more factors
$\;\;(-2) \times (-3) \times (-5)$
$=\; 6 \times (-5)$  (first two: same signs → +)
$=\; -30$  (different signs → −)
Shortcut for many factors
Count the negative signs. An even number of negatives gives a positive answer. An odd number of negatives gives a negative answer.

Practice

1.$(-8) \times 5 = \;?$
Answer:$-40$  (different signs → negative)
2.$(-9) \times (-7) = \;?$
Answer:$63$  (same signs → positive)
3.$(-1) \times (-2) \times (-3) \times (-4) = \;?$
Answer:$24$  (four negatives → even → positive)
4.$6 \times (-2) \times 3 = \;?$
Answer:$-36$  (one negative → odd → negative)

Dividing Integers

Good news — division uses the exact same sign rules as multiplication.

Sign 1Sign 2Quotient
(+)(+)(+)
(−)(−)(+)
(+)(−)(−)
(−)(+)(−)
Same idea
Same signs → positive.   Different signs → negative.
Example 1
$\;\;(-36) \div 4 = -9$  (different signs → negative)
Example 2
$\;\;(-72) \div (-9) = 8$  (both negative → positive)
Example 3
$\;\;\dfrac{45}{-5} = -9$  (different signs → negative)
Watch out — Zero
$0 \div n = 0$ for any nonzero integer $n$. But $n \div 0$ is undefined — you can never divide by zero.

Practice

1.$(-48) \div 6 = \;?$
Answer:$-8$  (different signs → negative)
2.$(-100) \div (-10) = \;?$
Answer:$10$  (same signs → positive)
3.$\dfrac{56}{-8} = \;?$
Answer:$-7$  (different signs → negative)
4.$0 \div (-15) = \;?$
Answer:$0$  (zero divided by anything nonzero is zero)
