Function Characteristics
📈 Growth vs. Decay
The base b determines everything. When b > 1 the function grows; when 0 < b < 1 it decays.
📈 GROWTH
Base: b > 1
Direction: Rises left → right
As x → +∞: y → +∞
As x → −∞: y → asymptote
Rate formula: b = 1 + r
Example: f(x) = 3(2)ˣ
📉 DECAY
Base: 0 < b < 1
Direction: Falls left → right
As x → +∞: y → asymptote
As x → −∞: y → +∞
Rate formula: b = 1 − r
Example: f(x) = 3(½)ˣ
GROWTH: f(x) = 3(2)ˣ
DECAY: f(x) = 3(½)ˣ
Quick Classifier — Type a base to check
Identifying Growth vs Decay — Method
1
Write the equation in standard form: f(x) = a · bˣ
2
Identify the base b (the number raised to the variable power)
3
If b > 1 → GROWTH ↑  |  If 0 < b < 1 → DECAY ↓
4
Find rate: Growth: r = b − 1  |  Decay: r = 1 − b  (then multiply by 100 for %)

The Key Rule
If bx = by → then x = y  (when b > 0 and b ≠ 1)
Rewrite both sides with the same base, then the exponents must be equal — or the same direction for inequalities.

🔢 Geometric Sequences
A geometric sequence multiplies by a constant common ratio r at each step.
Recursive Formula
aₙ = r · aₙ₋₁
Also state a₁ (the first term)
Explicit (Closed) Formula
aₙ = a₁ · r^(n−1)
Find any term directly
💡 To find r: divide any term by the previous one — r = aₙ ÷ aₙ₋₁
